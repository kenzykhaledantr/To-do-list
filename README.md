# To-Do List App

A simple and clean To-Do List application to help you manage your daily tasks.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks

## How to Run
1. Clone the repository:
```
   git clone https://github.com/kenzykhaledantr/To-do-list.git
```
2. Open the project folder
3. Run the app
